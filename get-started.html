How to add it to your site
Squarespace
  1. Settings → Advanced → Code Injection
  2. Paste the code into the Footer field
  3. Save — it's live in 60 seconds
Wix
  1. Settings → Custom Code → + Add Custom Code
  2. Paste the code, set "Add code to" → All pages
  3. Set "Place code in" → Body - end
WordPress
  1. Install "Insert Headers and Footers" plugin
  2. Settings → Insert Headers and Footers
  3. Paste in Scripts in Footer, save
Shopify
  1. Online Store → Themes → Edit code
  2. Open theme.liquid
  3. Paste before </body>
Other / Custom HTML
  1. Open your site's main HTML file
  2. Paste the code just before the closing </body> tag
  3. Save and deploy
